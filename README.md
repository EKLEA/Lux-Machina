«Lux Machina» – казуальная стратегия в реальном времени, где игрок
управляет механическим городом-крепостью. Ключевой ресурс –
«Жизненная Энергия», которую вырабатывает ядро-генератор. Игрок
устанавливает оптические линзы, чтобы перенаправлять потоки энергии к
критическим системам:
• Заводам, производящим компоненты для строительства укреплений,
боеприпасов для обороны и деталей для починки ядра;
• Оборонным сооружениям, обеспечивающим защиту от атак.
Главная цель – восстановить ядро, чтобы отпугнуть врагов навсегда. Каждая
волна атак заставляет принимать быстрое решение: направить энергию на
немедленную оборону или на производство для долгосрочного выживания.
